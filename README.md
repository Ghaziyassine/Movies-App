## Running the project

Clone the repo: <br>
`git clone git@github.com:arthurbz/tmdb-app.git`

Then cd into it:<br>
`cd tmdb-app`

Create a .env file with your TMDB API Key:<br>
`echo API_KEY={API_KEY} >> .env`

Install dependencies:<br>
`npm i`

Run ðŸ¤ <br>
`npm start`
